LobbyManager Fields

The LobbyManager type exposes the following members.

Fields
  NameDescription
m_BackButton
Returns the user to the Connection Menu panel.
m_ChosenMatch
The match that a user selected to join
m_ConnectButton
This button is what allows the player to select their username - connecting them to the GameSparks API and moves them to the connection menu panel. It is a child of the Main Menu Panel.
m_ConnectionMenuPanel
This panel appears after the user has created a user name and connected. It gives the user the option to host a session or look for one.
m_ConnectionStatusText
This text displays the user's connection status with GameSparks API while they are in the LobbyScene. It is a child of the Info Text Panel.
m_FindButton
This button launches the Room View panel where users can see all the games currently being hosted. ///
m_HostButton
This button launches the host menu panel for the user. From here they can set up the session they want to host.
m_HostErrorText
Displays the error that occurred when attempting to host a match.
m_HostMenuPanel
This is the panel that appears when the user chooses to host a session. It gives the user the option to select the match type and maximum player count
m_HostSelectionDropDown
The dropdown menu that the host can select to determine what match they will be hosting.
m_InfoTextPanel
This panel stays visible throughout the lobby, displaying the user's username and connection status
m_JoinMatchButton
Using data gathered from the match clicked on, the user joins a match.
m_JoinMatchErrorText
Display any errors when attempting to join a match
m_LobbyScreen
The screen the player sees after they have joined a matc.h
m_MainMenuPanel
This panel is the first one to displayed on scene load. It allows the user to create a username and connect
m_MatchButtonPrefab
The prefab that will spawn buttons containing match info for users to join a match with.
m_MatchNameText
The name of the match displayed to the user on the lobby screen.
m_MatchShortCodes
List of match short codes gathered from the GameSparks API. These are used to host and find matches.
m_MaxPlayerCount
String containing the max player count for a match
m_OldMatchButtonPosition
The position of the old match button
m_PlayerCountText
The amount of players out of total players in a match - displayed to the user on the lobby screen.
m_PlayerListText
List of all of the players on the lobby screen.
m_QuickConnect
Informs this class if QuickConnect is being used or not to set up a connection, adjusting what is displayed to the user based on this value
m_ReadyButton
The button that player pushes to indicate they are ready for the match to begin.
m_RefreshButton
Refreshes the search for finding a room (sends the request again).
m_RoomName
The name of the room that users should join
m_RoomNameInputField
Allows the user to name their match/room.
m_RoomViewPanel
This is the panel that appears when the user chooses to find a session. It displays all of the matches currently being hosted and allows the player to join any of the ones available.
m_SearchMatchErrorText
Displays the error the at occurred when attempting to search for a match.
m_StartHostingButton
This button starts hosting a match for the selected match in m_HostSelectionDropDown.
m_StartingScene
The non-static string that represents scene name of the scene to load after all players ready up. Used to set its static counter part since it can be seen in the Unity Editor and static variables cannot.
m_StaticStartingScene
Static string that represents the scene name of the scene to load after all players ready up. Static so that it can be seen in other classes without having to have an object of this class instantiated
m_TempRTSessionInfo
Creates a temporary RTSession which will hold data to be used in the real RTSession when the users ready up - thus launching the real RTSession
m_UsernameInput
This input field is where the user inputs their username. It is a child of the Main Menu Panel.
m_UserNameText
This text displays the user's username while in the LobbyScene. This is a child of the Info Text Panel.
