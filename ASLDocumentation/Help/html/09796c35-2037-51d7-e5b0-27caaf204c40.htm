ASL Namespace

All of the code relating to the Augmented Space Library (ASL).
Classes
  ClassDescription
ASLHelper
Provides functions pertaining to ASL to be called by the user but not linked to any specific object
ASLObject
ASLObject: ASLObject Partial Class containing all of the functions and variables relating to server actions - actions that affect all players instead of just a single player. use this class to communicate object information to other players.
GameController
GameController: Part of the GameController class that handles everything specific to GameController or used by just the GameController class
GameSparksManager
This class handles all of the incoming packets - directing them where they need to go. It persists between scene loads and is accessed through a singleton variable.
JSONFoundMatchInfoConverter
Used to convert match found information from the GameSparks Servers into a readable format to display to the player
JSONMatchNameConverter
Used to convert the name of a match from the GameSparks Server into a readable format to help display information to the player
LobbyManager
This class connects us to other players via functions in GameSparksManager and itself regardless if QuickConnect is being used or not
QuickConnect
Class used to quickly connect to a match for testing proposes
RTPlayer
Contains information about a player connected to GameSparks
RTSessionInfo
Holds a RealTime session info to be used when all players are ready
