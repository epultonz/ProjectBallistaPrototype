SendFloat4_Example
An example of how to use SendFloat4. While you can only send 4 values at a time, if you need your object to 5+ different values, at different times, you can do this via a switch/case, like this example shows. If you are using 4 of less values Then there is no need to use the switch example, but instead to just assign your values as you need them.
TransformObject_Example
Example of how to perform the various Transforms (position, rotation, scale) on an object. As well as some examples of what to do for additive style movement
